# 🎓 Student Manager App

## Stack
- **Backend**: Python + Flask (2 libraries only)
- **Frontend**: React + Vite (minimal)

---

## ▶️ Run Backend

```bash
cd backend
pip install -r requirements.txt
python app.py
```
Runs on: http://localhost:5000

---

## ▶️ Run Frontend

```bash
cd frontend
npm install
npm run dev
```
Runs on: http://localhost:5173

---

## Features
- ✅ Register / Login
- ✅ Add Student (Name, Age, Grade, Email)
- ✅ Edit Student
- ✅ Delete Student
- ✅ Data saved in `data.json`
